## Design Context

### Users
This is a personal blog and tool workspace for a creator who writes, studies, stores knowledge, and uses AI to help with planning, reading, and lightweight research. The main user is often the owner, with occasional public visitors.

### Brand Personality
Quiet, thoughtful, and crafted. The interface should feel like a private study desk: useful first, but with enough visual character to make long sessions pleasant.

### Aesthetic Direction
Use refined product UI with soft literary details. Prefer calm surfaces, strong information hierarchy, and restrained decorative touches. Avoid generic dashboard sameness, heavy neon AI styling, and decorative clutter.

### Design Principles
- Make the first screen usable, not promotional.
- Keep workflows clear enough for repeated personal use.
- Use visual richness to support focus, not to compete with content.
- Treat AI output as assisted interpretation, with visible sources and user control.
- Keep Chinese content comfortable to read, with English labels only when they help orientation.
